# INSTALLATION
1. Install dependencies:
- `numpy`
- `scipy`
- `numba` 
- `healpy` (`conda install -c conda-forge healpy`)
- `fastprogress` (`pip install fastprogress`)

2. Enter root directory of project 'proj/` directory. This file has the structure
	proj/
├─ axion/
├─ LICENSE
├─ MANIFEST.in
├─ README.md
├─ setup.cfg

3. Run `pip install .`
